Introduction

The main scope of the FRIC project is to build and deploy a library for fractal generation. In this file you will find out how to make best use of the library and the platform we provide.

If you prefer a visual demonstration of the project in action, you can watch this video.

Structure

We will break down the guide into two parts:

The platform

  1. Navigation around the platform is easily done using the nav-bar at the top of the page.
  2. On the browse page the user can admire other's people fractals or unpublish his fractals that he's not entirely proud of.
  3. On the build page the user can generate fractals using the side menu and after he is satisfied with his work he can publish it on the platform for other's to see.
  4. Lastly there is the page used to generate keys, which we will talk about in the library section.

The library

  1. First of all, to be able to use the library, one must first generate a token on the platform. This is just two clicks away after creating an account.

  2. Now you have a brand new key you can use to generate 300 fractals/day.
  3. Using the key you can now initialize the fractal builder in your application.
  4. Finally you can generate your awesome fractal with your own definition.
  5. This will return an object containing a link to the fractal you just generated.
